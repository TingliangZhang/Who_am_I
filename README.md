张庭梁个人简介 中文版
===



张庭梁，于2020年本科获清华大学电气工程及其自动化工程学士学位，清华大学美术学院信息艺术设计交叉学科研究生，目前在清华大学未来实验室。入选星火计划十二期（“科技创新，星火燎原”清华大学学生创新人才培养计划）。曾任清华大学“未来通信”学生科技兴趣团队理事长，清华大学“天空工场”学生科技兴趣团队“未来航空队”队长。2018年暑假在劳伦斯 - 伯克利国家实验室和加州大学伯克利分校 MSC Lab暑期研修。曾获清华大学“挑战杯”特奖。

张庭梁的主要研究方向为机器人、实体交互设计、信息艺术设计。涉足UX用户体验设计

## 部分科研项目摘要(均为一作或共一)

### Misaka：用于人机交互研究的集群软硬件开发平台

本作品介绍了一个用于算法评估的可视化测试平台，同时也是用于开发桌面实体集群交互界面的可扩展软硬件开源平台。包括一组定制设计的3全向轮机器人（每个直径10厘米）、配套的软件以及开发者版本PCB。从硬件和通信协议层面实现了软硬件的完全去中心化，为共识算法等提供了完备的硬件测试和可视化平台。

创新点主要有：

-  将分布式算法用于电力市场经济调度中，去中心化可以增强电力系统的可靠性。
-  使用集群可视化平台探索新的展示算法的方式。
-  开发了一套完备的通信架构用于集群通信开发和测试。
-  较为成熟的集成化移动平台集群系统为人机交互创造了无限的可能性。

### SUSI：一种自我觉察、辅助治疗和寻求帮助的工具

SUSI是一个预防和协助心境障碍患者进行自我觉察、辅助治疗和寻求帮助的安全、友好、便携的工具。它鼓励正在经历情绪问题的人们在合适方式下坦露和面对自己内心的脆弱，从而帮助他们发泄和进行认知重评。SUSI可以附着在手机背面，使用时的姿势类似于打电话，即使在公共场合也不会引起不必要的尴尬。SUSI是一个离线工具，所以用户不需要担心数据的泄露问题。通过SUSI进行记录，用户可以查看历史记录的情绪数据统计分析来认识自我的情绪变化趋势，通过词频分析帮助用户找到引起个人情绪变化的原因。此外，通过SUSI，使用者可以在需要帮助的时候选择合适的被信任的分享对象，用自动生成的针对不同亲密关系的适当语言进行分享，从而在避免重复叙述产生二次伤害的前提下，提供更加有效的叙述信息，以便寻求他人帮助。我们的根本目的不是解决患者的当下之难，而是通过一种自我觉察的方式，帮助使用者重新回到正常生活。

### 基于 Vive VR 的遥操作机器人

随着机器人技术的发展，机器人的应用场景越来越多。但是目前自动控制机器人尚不能执行多数复杂任务，特别是抢险救灾等需要随机应变的任务，这种情况下需要遥操作机器人。

目前机械臂远程控制普遍采用手柄或键盘控制方式，且监控方式普遍为摄像头图像显示在监视器上，与现场操作差别很大。

为此，我们开发了四个硬件架构大不相同的迭代版本的用 VR 设备远程控制机器人的系统以及配套的三维实时场景采集及图传系统，使操作者能有身临其境的操作体验，大幅降低成本的同时减小了延迟

### NANO Robot -- 基于 Nvidia Jetson NANO 的人工智能教育套件  

针对目前市场上人工智能 STEAM 高端教育资源短缺的情况，开发了一套以 Nvidia Jetson NANO 为核心控制，定制版本 Ubuntu1804LTS 系统， OpenCV/TensorFlow/CUDA 等为基础的人工智能教育套件。其中典型的一 款产品为智能小车套件，能够让青少年通过简单的操作就能实现车道线检测，视觉避障，场景识别，人脸 ID，图像分割等功能

将命令行代码编程与可视化图形编程相结合，通过 Web 浏览器进行交互式编程，拥有从基本运动到人工智能的教程，简单易用，深入浅出。

### 智能电表--基于NILMTK的非侵入式负荷分解装置

目前具有非侵入式负荷监控功能的智能电表还没有实用的方案，主要原因是电器种类太多功率特性差异不明显，所以绝大多数解决方案是以在线神经网络分析为主，即将全部采集到的数据，上传到云服务器，在服务器上 用 GPU/CPU 运算后反馈给用户或电能提供商。这样不仅需要给产品加上昂贵的大带宽通信解决方案，而且占用了很大的算力，用户的数据安全性也存在一定风险。

本项目提出了一种不依赖神经网络的算法，可以在廉价节能的嵌入式芯片上运行。同时，设计了一套完整的智能电表解决方案，将板上系统完美融合到电表之中，所有数据都在电表中完成处理，只上传需要的参数给电网公司，用户可以决定是否出售其他数据，涉及敏感信息的结果只在本地储存。

可以分辨电磁炉、空调、微波炉等大功率用电器每天开启的时间段，可以为家庭需求侧响应提供数据基础。

### 基于 PlaceCNN 的无障碍出行灯光提醒装置

腿脚不便人士出行时由于移动速度等问题，会存在一些安全问题，通过使用 PlaceCNN 训练得到的神经网络，可以得到当前场景信息，从而有不同的声光提示策略。

### 基于柔性 LED 阵列的穿戴产品的研究

与美院合作项目。柔性 LED 作为一类可弯曲、鲁棒性高的特种电路，为艺术表达提供了一种新的形式的可能性，本作品则提出了一种以普通布料为基底的柔性 LED 阵列的解决方案。

### 与新媒体团队及美院合作项目“自动光绘无人机”

无人机上携带高亮度的 RGB-LED，在 GPS 和光流的辅助下，以可编程的灯光和轨迹飞行，地面的相机长曝光拍摄，得到空中的光绘作品。

### Coilgun:多级高斯线圈电磁加速装置

通过电磁相互作用力加速载荷的装置，线圈内在极短的时间内流过 100A 的大电流，产生强磁场加速小铁球，通过光电门检测到小铁球通过线圈中点时断电， 通过多级线圈加速可以使铁磁性介质获得很大能量。
